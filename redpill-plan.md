# План разработки мультиагентной системы RedPill

## Архитектура
Система строится по паттерну Supervisor + Workers на базе LangChain.

### Компоненты
1. **Supervisor (Главный Агент)**
   - Координация работы суб-агентов.
   - Хранение общего состояния и торгового плана.
2. **Analyst (Суб-агент)**
   - Инструменты: `get_klines`, `get_ticker`, `get_positions`, `calculate_math`.
   - Анализ: Свечной анализ (5x15m), Pivot Points, Z-Score.
3. **Money Manager (Суб-агент)**
   - Инструменты: `get_balance`, `get_history`.
   - Логика: Расчет объема сделки и уровней риска.
4. **Trader (Суб-агент)**
   - Инструменты: `place_order`, `set_sl_tp`, `cancel_order`.
   - Задача: Исполнение торгового плана.

## Шаги реализации
1. Инициализация проекта (requirements, core config).
2. Реализация инструментов Bybit.
3. Создание промптов и суб-агентов.
4. Сборка системы Supervisor.
5. Настройка Docker.
